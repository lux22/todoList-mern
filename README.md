# TodoList-mern

A todo list MERN stack application with CRUD operations. User will be able to create task and also upload images for each task created.

## Features

- Switch task between completed and pending state.
- Upload images
- Edit and delete task.

## Tech (Client)

- React Js
- Redux
- Redux Thunk
- Material-Ui

## Tech (Server/Database)

- Mongodb
- Mongoose
- Node
- express

## Installation

Install the dependencies and devDependencies and start the server.

**Client Side**

```sh
cd client
npm install
npm start
```

**Server Side**

```sh
cd server
npm install
npm start
```
